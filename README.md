# 村庄保卫战
## 1.关于项目
• 项目使用 C++ 语言并利用 SDL 程序库进行开发，引入硬件加速支持的 2D渲染的实现，提高游戏性能。
• 通过存储瓦片地图和游戏中敌人等单位的信息，并对这两部分采用
CSV/JSON 数据格式解析加载到游戏中。
• 完成了对敌人的高性能的寻路算法和防御塔的优先索敌策略及升级和属性变更功能。
• 通过波次控制游戏的导演系统，完善了 GamePlay 功能，使其看起来更符合 PVE 游戏逻辑。
